# 🚀 AWS DevOps Production Kit

A production-ready AWS DevOps starter kit using Terraform, Docker, GitHub Actions, and Auto Scaling — without Kubernetes.

---

## 🔥 Features

- Scalable infrastructure using **Application Load Balancer (ALB)**
- Auto Scaling Group (ASG) for high availability
- Dockerized Flask application
- CI/CD pipeline using GitHub Actions
- CloudWatch monitoring setup
- Modular Terraform structure
- Beginner-friendly and production-style architecture

---

## 🧰 Tech Stack

- AWS (EC2, ALB, Auto Scaling, VPC, CloudWatch)
- Terraform
- Docker
- GitHub Actions
